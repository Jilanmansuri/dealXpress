# DealXpress Frontend

This is the React frontend for the DealXpress application.

## Tech Stack
- **Framework**: [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **State Management**: [Redux Toolkit](https://redux-toolkit.js.org/)
- **API Client**: [Axios](https://axios-http.com/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **UI Components**: [Material UI (MUI)](https://mui.com/)
- **Routing**: [React Router Dom](https://reactrouter.com/)

## Project Structure
The `src` directory follows a scalable architecture:
- `components/`: Reusable UI components.
- `pages/`: Full page components.
- `features/`: Feature-specific logic and states.
- `hooks/`: Custom React hooks.
- `services/`: API clients (Axios) and external services.
- `utils/`: Helper functions and constants.
- `layouts/`: Layout components (Navbar, Sidebar, etc.).
- `routes/`: Route definitions and protection.

## Getting Started

### Prerequisites
- Node.js (v16+)
- npm or yarn

### Installation
```bash
# Navigate to the frontend directory
cd Frontend

# Install dependencies
npm install
```

### Development
```bash
# Start the development server
npm run dev
```

### Build
```bash
# Create a production build
npm run build
```
